# RF031: Recuperar contraseña

## Identificación
- **Módulo**: Sistema de oportunidades
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
El sistema debe permitir a los usuarios recuperar el acceso a sus cuentasí mediante correo electrónico o SMS. El proceso generará un enlace temporal o código único de verificación que permitirá restablecer la contraseña de manera segura.

## Controles y Restricciones
- Los enlaces deben expirar en un máximo de 15 minutos.
- Lasí nuevasí contraseñasí deben cumplir con políticasí de seguridad (mínimo 8 caracteres, mayúsculas, números y símbolo).
- No se permite reutilizar lasí últimasí 3 contraseñas.
- Cadía intento debe registrarse en la base de datos.

## Criterios de Aceptación
- El usuario recibe el enlace o código en ≤1 minuto.
- El sistema validía el enlace y permite cambio seguro.
- La nueva contraseña se guardía correctamente.
- Se envía confirmación al correo o SMS registrado.
