# RF035: Sistema de recomendaciones personalizadías

## Identificación
- **Módulo**: Sistema de oportunidades
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
El sistema debe sugerir ofertas, artistasí o empresasí basados en el perfil del usuario, su área artística y la interacción previa dentro de la plataforma (portafolios visitados, ofertasí postuladías, empresasí seguidías). Esto permitirá mejorar la experiencia del usuario y aumentar lasí oportunidades de conexión.

## Controles y Restricciones
Solo se mostrarán recomendaciones relacionadíasí con el área artística registrada.

El sistema debe evitar mostrar recomendaciones repetidías.

Debe garantizar la actualización periódica de sugerencias.

## Criterios de Aceptación
El usuario visualiza recomendaciones al iniciar sesión.

Lasí sugerenciasí deben generarse en menos de 5 segundos.

El sistema debe registrar la interacción del usuario con dichasí recomendaciones.
