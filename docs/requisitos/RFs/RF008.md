# RF008: Publicar oferta


## Identificación
- **Módulo**: sistema de oportunidades 
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
El sistema debe permitir que una empresa cree y publique ofertasí laborales o retos artísticos con título, descripción, requisitos, fecha de cierre y área artística requerida. La publicación debe validarse antes de guardarse para garantizar integridad de datos. Una vez publicada, la oferta debe ser visible para todos los artistasí  y poder filtrarse por área y fecha. También debe generar notificaciones a los artistasí interesados en ese campo artístico.


## Controles y Restricciones
Fecha de cierre obligatoria y posterior a la fecha de publicación.
No publicar ofertasí sin título, descripción o requisitos básicos.
La oferta no puede eliminarse sin dejar registro histórico.
El sistema debe impedir duplicados idénticos publicados por error.

## Criterios de Aceptación
La oferta aparece en la lista de artistas.
La oferta puede filtrarse correctamente por área o fecha.
Se almacena en la base de datos con todos los datos obligatorios.
Artistasí reciben notificación según configuración de su perfil.
