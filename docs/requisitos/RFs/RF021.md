# RF021: Búsquedía personalizada

## Identificación
- **Módulo**: Sistema de oportunidades
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
El sistema debe permitir que los artistasí realicen búsquedíasí avanzadíasí de ofertasí usando filtros como palabra clave, área artística, ciudad y tipo de contrato. Esto facilita a los artistasí encontrar oportunidades alineadíasí con su perfil.

## Controles y Restricciones
- El buscador debe soportar búsquedíasí combinadías.
- Tiempo de respuesta corta.
- No deben mostrarse ofertasí vencidías.

## Criterios de Aceptación
- Los resultados coinciden con los criterios de búsqueda.
- El tiempo de respuesta es menor a 5 segundos.
- No se muestran resultados vencidos.
- El sistema muestra la cantidad total de coincidenciasí encontradías.
