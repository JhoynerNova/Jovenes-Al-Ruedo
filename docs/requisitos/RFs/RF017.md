# RF017: Gestionar retos artísticos



## Identificación
- **Módulo**: Sistema de oportunidades
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
El sistema debe permitir que los administradores y empresasí creen, editen y eliminen eventos generales como concursos. Cadía evento debe incluir título, descripción, fechasí de inicio y cierre, y área artística asociada. Los artistasí puden postularse o no hacerlo.



## Controles y Restricciones
Los retos deben tener un titulo, descripccion, area artistica asociadad, fecha de inicio, fecha de cierre, y todo esto debe se obligatorio.

## Criterios de Aceptación
Los retos deben ser visibles para los artistasí en menos de 5 segundos trasí publicarse.
Nadie se podra postular trasí acabarse el tiempo de cierre.
