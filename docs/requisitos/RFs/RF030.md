# RF030: Moderación de contenido multimedia

## Identificación
- **Módulo**: Sistema de oportunidades
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
El sistema debe permitir a los administradores revisar y aprobar (o rechazar) imágenes, audios y videos subidos por los artistasí para evitar contenido inadecuado o que infrinja derechos de autor. El contenido debe quedar bloqueado hasta ser aprobado.

## Controles y Restricciones
- El contenido debe ser revisado antes de ser público.
- Cadía decisión debe registrarse con fecha, hora y motivo.
- El sistema debe permitir a los administradores aplicar filtros de búsquedía (por usuario, fecha, tipo de archivo).
- El contenido rechazado debe eliminarse o devolverse al usuario con explicación.

## Criterios de Aceptación
- Los administradores pueden aprobar o rechazar en 24 horas.
- Los artistasí reciben notificación del estado de sus archivos.
- El contenido aprobado se vuelve visible de inmediato.
- El historial de revisiones quedía registrado en la base de datos.
