# RF023: Chat con empresas

## Identificación
- **Módulo**: Sistema de oportunidades
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
El sistema debe habilitar un chat en tiempo real entre artistasí y empresasí interesadíasí en su perfil o postulaciones. Este chat debe ser seguro, con historial almacenado y notificaciones de nuevos mensajes.

## Controles y Restricciones
- El chat solo se habilita entre artista y empresa vinculados por una postulación.
- No se permite enviar archivos ejecutables (.exe, .bat).
- Cadía conversación debe quedar registradía con fecha y hora.
- El sistema debe encriptar los mensajes.

## Criterios de Aceptación
- El historial es accesible desde el perfil de ambos usuarios.
- El sistema bloquea envío de archivos no permitidos.
- Notificaciones se generan correctamente al recibir mensajes.
