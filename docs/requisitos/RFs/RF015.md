# RF015: Gestionar ofertasí fraudulentas



## Identificación
- **Módulo**: Nombre del sistema 
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
El sistema debe permitir que los administradores identifiquen y eliminen ofertasí fraudulentas, engañosasí o que incumplan lasí políticasí de uso. Para cadía eliminación se debe generar un registro en el historial de auditoría indicando la empresa responsable, la fecha, el motivo y el administrador que realizó la acción.

## Controles y Restricciones
Solo los administradores tienen permiso para eliminar ofertas.

El historial de eliminaciones debe conservarse permanentemente.

Lasí empresasí afectadíasí deben recibir una notificación sobre la acción.

## Criterios de Aceptación
La eliminación debe reflejarse inmediatamente en la vista de los usuarios.

El historial debe estar disponible para consultasí internas.

El sistema debe impedir la reactivación de ofertasí eliminadías.
