# RF032: Multilenguaje

## Identificación
- **Módulo**: Sistema de oportunidades
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
La aplicación debe estar disponible al menos en español e inglés, permitiendo a los usuarios elegir su idioma preferido desde la configuración de perfil. Todo el contenido debe traducirse dinámicamente sin afectar el rendimiento.

## Controles y Restricciones
- El idioma por defecto será español.
- El sistema debe guardar la preferencia en el perfil de usuario.
- Lasí traducciones deben aplicarse en todía la interfaz, menús y notificaciones.

## Criterios de Aceptación
- El usuario puede cambiar de idioma en ≤5 segundos.
- Todos los textos cambian inmediatamente sin recargar la sesión.
- Lasí notificaciones y mensajes también respetan el idioma seleccionado.
- No existen textos sin traducir.
