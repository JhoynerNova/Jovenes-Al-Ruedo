# RF039: Recomendaciones entre usuarios

## Identificación
- **Módulo**: Sistema de oportunidades
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción

Los usuarios (empresasí o artistas) podrán recomendar perfiles de otros artistasí o empresasí dentro de la plataforma, generando confianza y visibilidad.


## Controles y Restricciones
Solo se puede recomendar a usuarios activos.

Cadía usuario podrá recomendar máximo 1 vez a otro usuario.

Lasí recomendaciones deben ser visibles en el perfil recomendado.

## Criterios de Aceptación
El usuario recibe notificación cuando es recomendado.

El perfil debe mostrar el número total de recomendaciones recibidías.

Lasí recomendaciones deben poder ser eliminadíasí por el administrador en caso de abuso.
