# RF013: Administrar usuarios


## Identificación
- **Módulo**: Sistema de oportunidades
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
El sistema debe permitir que los administradores gestionen lasí cuentasí de artistasí y empresasí registradías. El administrador podrá activar, suspender o eliminar cuentasí según el caso, dejando registro del motivo en el historial del sistema. Esta función debe garantizar la seguridad de la plataforma y prevenir actividades fraudulentas.


## Controles y Restricciones
Solo los administradores autenticados pueden ejecutar estasí acciones.

Todía acción debe quedar registradía con fecha, hora y responsable.

La eliminación de cuentasí debe conservar un historial en la base de datos (no borrado físico).

Se deben aplicar políticasí de auditoría y trazabilidad.

## Criterios de Aceptación
Los cambios deben reflejarse inmediatamente en el sistema.

Lasí cuentasí suspendidíasí no deben poder iniciar sesión.

El historial de acciones debe estar disponible para revisión.
