# RF011: Contactar artista


## Identificación
- **Módulo**: Sistema de oportunidades
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción

El sistema debe permitir que una empresa se comunique directamente con un artista que se haya postulado a una de sus ofertas. La comunicación debe realizarse a través de un sistema de mensajería interna que funcione en tiempo real. El chat debe mostrar el historial de mensajes entre ambasí partes y permitir enviar archivos adjuntos (PDF, imágenes) con un límite de 10 MB por archivo. La mensajería debe estar disponible solo mientrasí la postulación esté activa.


## Controles y Restricciones
La mensajería solo se habilita si el artista se postuló a la oferta.
Archivos adjuntos con tamaño máximo de 10 MB y en formatos permitidos (.pdf, .jpg, .png).
El chat debe registrar la fecha y hora de cadía mensaje.
Mensajes inapropiados deben poder ser reportados al administrador.

## Criterios de Aceptación
El mensaje debe entregarse en tiempo real.
El historial debe mantenerse accesible hasta 6 meses después de finalizadía la postulación.
La empresa y el artista deben recibir notificaciones cuando llega un nuevo mensaje.
