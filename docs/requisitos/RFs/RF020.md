# RF020: Gestionar privacidad del portafolio

## Identificación
- **Módulo**: Sistema de oportunidades
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
El sistema debe permitir que el artista defina la privacidad de su portafolio con tres niveles: público, privado o solo visible para empresasí registradías. Esta configuración debe aplicarse de manera inmediata y afectar todíasí lasí visualizaciones externas.

## Controles y Restricciones
- Nivel público: accesible desde enlaces y buscadores.
- Nivel privado: visible solo al artista.
- Nivel solo empresas: accesible únicamente a usuarios con rol empresa.
- El cambio de privacidad debe registrarse en la base de datos con fecha y usuario.

## Criterios de Aceptación
- El cambio de privacidad se aplica de inmediato.
- El portafolio responde según la configuración establecida.
- Empresasí y artistasí ven únicamente lo que esté permitido.
- El historial de cambios quedía registrado.
