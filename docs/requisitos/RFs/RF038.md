# RF038: Pasarela de pagos

## Identificación
- **Módulo**: Sistema de oportunidades
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
El sistema debe integrar métodos de pago electrónicos para permitir que empresasí realicen pagos a artistasí por contrataciones o servicios. También debe permitir anticipos o depósitos de seguridad.
Controles y restricciones:

## Controles y Restricciones
Compatible con plataformasí de pago nacionales (ej. PSE, Nequi, Daviplata) e internacionales (PayPal).

El sistema debe registrar todíasí lasí transacciones en un historial.

Los pagos deben realizarse con cifrado seguro.

## Criterios de Aceptación
El artista recibe confirmación de pago en tiempo real.

La empresa puede descargar comprobantes en PDF.

Todíasí lasí transacciones deben quedar en la base de datos.
