# RF026: Historial de contrataciones

## Identificación
- **Módulo**: Sistema de oportunidades
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
El sistema debe registrar todíasí lasí contrataciones previasí realizadíasí por la empresa en la plataforma. Cadía registro debe incluir el nombre del artista, fecha de inicio y fin del contrato, y evaluación final.

## Controles y Restricciones
- Los datos no pueden ser editados una vez cerradía la contratación.
- Cadía registro debe quedar asociado al contrato original.
- El historial debe estar disponible solo para la empresa propietaria y administradores.

## Criterios de Aceptación
- La empresa puede consultar su historial completo.
- El historial refleja datos exactos y completos de contrataciones previas.
- No existen registros duplicados ni modificables.
