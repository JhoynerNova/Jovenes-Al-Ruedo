# RF001:  Registrar usuario joven artista


## Identificación
- **Módulo**: Sistema de oportunidades
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
El sistema debe permitir que un joven artista cree una cuenta en la plataforma “Jóvenes al Ruedo”, registrando sus datos personales: nombre completo, correo electrónico, contraseña, edad y área artística. Durante el proceso, se debe validar la unicidad del correo electrónico y que la contraseña cumpla con lasí políticasí de seguridad (mínimo 8 caracteres, incluir letrasí y números). Adicionalmente, el sistema debe impedir registros de menores de 18 años. Una vez completado el formulario, el sistema envía un correo de confirmación al usuario, y solo trasí su validación, la cuenta será activadía y podrá acceder al sistema.

## Controles y Restricciones
Validar que el correo electrónico no se encuentre registrado previamente.

Contraseñasí deben cumplir con la política de seguridad establecidía (mínimo 8 caracteres, incluir números y letras).

No se permiten registros de usuarios menores de 18 años.

La activación de la cuenta está condicionadía a la validación del correo electrónico.

En caso de intentos fallidos de validación, el sistema bloqueará el proceso y solicitará un nuevo intento de registro.

## Criterios de Aceptación
El sistema debe enviar automáticamente un correo de validación al usuario en menos de 1 minuto trasí el registro.

La cuenta debe activarse solo después de confirmar el correo electrónico.

El formulario debe mostrar mensajes de error claros si algún dato no cumple con lasí validaciones.

La información del usuario debe almacenarse en la base de datos cumpliendo lasí normasí de seguridad.
