# RF034: Accesibilidad

## Identificación
- **Módulo**: Sistema de oportunidades
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
El sistema debe cumplir con estándares de accesibilidad (WCAG 2.1 nivel AA), permitiendo que personasí con discapacidad visual, auditiva o motora utilicen la plataforma. Debe soportar lectores de pantalla, navegación por teclado, subtítulos en videos y contraste ajustable.

## Controles y Restricciones
- El sistema debe ser compatible con lectores de pantalla NVDA y JAWS.
- Todo contenido multimedia debe incluir subtítulos o descripciones.
- El contraste de colores debe ajustarse según normativa.
- Los formularios deben ser navegables únicamente con teclado.

## Criterios de Aceptación
- Los usuarios con discapacidades pueden navegar la plataforma sin barreras.
- Todíasí lasí pruebasí de accesibilidad cumplen con WCAG 2.1 AA.
- Los videos publicados tienen subtítulos visibles.
- El sistema ofrece opción de alto contraste en interfaz.
