# RF007: 
Registrar empresa


## Identificación
- **Módulo**: sistema de oportunidades
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
El sistema debe permitir a lasí empresasí registrarse ingresando su información básica como nombre, correo electrónico, sector, logo y NIT. Durante el registro, se debe validar que el correo y el NIT no estén previamente registrados. El logo debe cumplir con requisitos de formato y peso máximo. Una vez completado, la empresa recibirá un correo de confirmación con un enlace de validación que activará la cuenta para acceder al sistema, aparte se debe confimar que la empresa sea veridica.


## Controles y Restricciones
Validar correo único en la base de datos.
Validar NIT con estructura definida.
Logo 5 MB en formato .jpg o .png.
La cuenta quedía inactiva hasta validar el correo.
Si los datos no cumplen, el sistema debe impedir el registro.

## Criterios de Aceptación
La empresa recibe un correo de confirmación en 1 minuto trasí registro.
La cuenta se activa solo trasí validación por correo.
Los datos deben almacenarse íntegramente en la base de datos.
Si hay error, se muestran mensajes claros al usuario.
