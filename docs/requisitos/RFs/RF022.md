# RF022: Seguimiento de empresas

## Identificación
- **Módulo**: Sistema de oportunidades
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
El sistema debe permitir que los artistasí marquen empresasí como “favoritas” o “seguidías” para recibir notificaciones de sus nuevasí ofertas. Esta funcionalidad permitirá a los artistasí estar atentos a oportunidades de entidades que les interesen.

## Controles y Restricciones
- Cadía artista puede seguir hasta 100 empresas.
- Si una empresa es eliminada, desaparece de lasí listasí de seguimiento.
- El sistema debe notificar automáticamente nuevasí ofertasí de empresasí seguidías.
- Debe existir un historial de seguimiento por usuario.

## Criterios de Aceptación
- El artista puede seguir o dejar de seguir empresasí en cualquier momento.
- Lasí notificaciones se envían trasí publicación.
- El sistema refleja correctamente lasí empresasí seguidíasí en el perfil del artista.
- El historial se actualiza en tiempo real.
