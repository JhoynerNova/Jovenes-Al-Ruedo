# RF041: Modo exhibición

## Identificación
- **Módulo**: Sistema de oportunidades
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
El sistema debe permitir que los artistas activen un “modo galería” en sus portafolios, diseñado para ser mostrado en presentaciones, ferias o exposiciones culturales.

## Controles y Restricciones
Solo el propietario del portafolio puede activar/desactivar el modo galería.

El modo galería no debe permitir edición ni comentarios.

Debe ser compatible con pantallas de diferentes tamaños.

## Criterios de Aceptación
El artista puede activar la galería con un clic.

El portafolio se muestra en formato visual optimizado.

Los visitantes solo podrán ver, no editar ni descargar sin permiso.
