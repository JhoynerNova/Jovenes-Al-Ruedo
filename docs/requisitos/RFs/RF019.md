# RF019: Descargar portafolio en PDF

## Identificación
- **Módulo**: Sistema de oportunidades
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
El sistema debe permitir que el artista genere un archivo PDF con el contenido de su portafolio (imágenes, descripciones, datos de contacto). Este archivo podrá descargarse para uso personal, impresión o envío externo. Debe conservar un diseño profesional y ser coherente con el perfil en línea.

## Controles y Restricciones
- El PDF debe generarse en formato A4 con estructura uniforme.
- Tamaño máximo del archivo 20 MB.
- Debe incluir nombre, datos de contacto y fecha de generación.
- No se permiten descargasí si el portafolio está vacío.

## Criterios de Aceptación
- El archivo contiene todo el contenido del portafolio en orden lógico.
- El archivo es descargable en cualquier dispositivo.
- El documento conserva formato legible y profesional.
