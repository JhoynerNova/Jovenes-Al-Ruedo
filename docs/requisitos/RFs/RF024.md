# RF024: Destacar oferta

## Identificación
- **Módulo**: Sistema de oportunidades
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
El sistema debe permitir que una empresa destaque una oferta laboral o artística para que aparezca en primerasí posiciones y con resaltado visual. Esto busca mejorar la visibilidad de oportunidades importantes y fomentar la inversión en el sistema.

## Controles y Restricciones
- Solo empresasí con cuenta activa pueden destacar.
- El servicio puede estar asociado a un plan de pago.
- No se permite destacar ofertasí vencidías.
- El sistema debe registrar fecha y duración del destaque.

## Criterios de Aceptación
- La oferta destacadía aparece en lasí primerasí posiciones inmediatamente.
- El resaltado visual es visible para todos los artistas.
- El sistema refleja correctamente el estado “Destacada”.
