# RF002: Iniciar sesión


## Identificación
- **Módulo**: sistema de oportunidades
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
El sistema debe permitir que un usuario registrado accedía a su cuenta mediante correo electrónico y contraseña. Debe existir validación inmediata de credenciales. Si el usuario realiza más de tres intentos fallidos, el sistema bloqueará temporalmente el acceso para evitar intentos de intrusión. La sesión debe expirar automáticamente trasí 30 minutos de inactividad.


## Controles y Restricciones
Bloqueo automático trasí 3 intentos fallidos.

Sesión expira trasí 30 minutos de inactividad.

Autenticación segura con cifrado de contraseñas.

## Criterios de Aceptación
Acceso válido debe ocurrir en menos de 3 segundos.

En caso de credenciales incorrectas, mostrar mensaje claro.

El bloqueo debe levantarse solo con recuperación de contraseña.
