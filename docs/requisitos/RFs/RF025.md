# RF025: Guardar artistasí favoritos

## Identificación
- **Módulo**: Sistema de oportunidades
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
El sistema debe permitir a lasí empresasí marcar perfiles de artistasí como “favoritos” para revisarlos posteriormente. Esto permitirá llevar un registro rápido de posibles candidatos sin necesidad de revisar nuevamente todo el catálogo.

## Controles y Restricciones
- Cadía empresa puede guardar hasta 500 artistas.
- Si un artista elimina su perfil, desaparece de favoritos.
- La acción debe registrarse en la base de datos.
- Debe existir opción para eliminar de favoritos.

## Criterios de Aceptación
- El sistema guardía y muestra los artistasí favoritos de manera persistente.
- El historial de favoritos se actualiza en tiempo real.
- La empresa puede acceder a los perfiles guardados sin error.
