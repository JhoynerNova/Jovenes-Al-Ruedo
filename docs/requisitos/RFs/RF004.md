# RF004: Editar portafolio


## Identificación
- **Módulo**: Nombre del sistema de oportunidades 
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
El sistema debe permitir que un artista edite su portafolio personal, pudiendo modificar títulos, descripciones y reemplazar archivos cargados previamente. No se deben aceptar formatos inseguros ni modificaciones que puedan afectar la integridad del sistema.


## Controles y Restricciones
No se permiten formatos no soportados (.exe, .bat, .msi).

Validar tamaño máximo de archivos según políticasí del sistema.

Cambios deben quedar registrados en el historial de modificaciones.

## Criterios de Aceptación
Los cambios deben reflejarse inmediatamente en el perfil.

Los archivos reemplazados deben eliminarse de forma segura.

El historial de modificaciones debe registrar la fecha y usuario que realizó el cambio.
