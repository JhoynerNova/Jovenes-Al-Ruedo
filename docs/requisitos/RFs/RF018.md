# RF018: Compartir portafolio en redes sociales

## Identificación
- **Módulo**: Sistema de oportunidades
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
El sistema debe permitir que el artista comparta su portafolio en redes sociales como Facebook, Instagram, LinkedIn o mediante un enlace público. Al generar el enlace, el portafolio debe mantenerse sincronizado con los cambios realizados en la plataforma. Esta funcionalidad busca dar mayor visibilidad a los artistasí fuera del ecosistema interno.


## Controles y Restricciones
- El enlace generado debe ser único y seguro.
- Se debe controlar que solo los portafolios públicos o visibles para empresasí puedan compartirse.
- El enlace debe caducar si el portafolio se elimina o cambia a privado.
- El sistema debe registrar cadía acción de “compartir” en la base de datos.

## Criterios de Aceptación
- El artista puede compartir su portafolio.
- El enlace compartido muestra el contenido más actualizado del portafolio.
- Los cambios en visibilidad afectan automáticamente los enlaces previos.
