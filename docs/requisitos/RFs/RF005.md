# RF005: Postularse a oferta

## Identificación
- **Módulo**: sistema de oportunidades
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
El sistema debe permitir que un artista se postule a una oferta laboral o reto publicado por una empresa. Para ello, el artista debe tener al menos un portafolio en la plataforma. La postulación debe incluir un enlace a su portafolio y un mensaje opcional de presentación. El sistema debe validar que la oferta esté activa y que el artista no se haya postulado previamente. Una vez enviadía la postulación, la empresa debe recibir una notificación inmediata (en la aplicación y por correo electrónico) y la información debe almacenarse en la base de datos, quedando disponible para seguimiento y auditoría.



## Controles y Restricciones
Solo artistasí con portafolio publicado pueden postularse. No se permite postularse a ofertasí cerradíasí o vencidías.
El sistema debe impedir postulaciones duplicadíasí a la misma oferta por el mismo usuario.
 La base de datos debe registrar fecha y hora de cadía postulación, así como su estado inicial (Pendiente).

## Criterios de Aceptación
La empresa debe recibir la postulación en tiempo real.
El artista debe visualizar un mensaje de confirmación trasí completar el proceso.El registro de la postulación debe guardarse íntegramente en la base de datos.
Si la postulación es rechazadía por duplicado o cierre, el sistema debe mostrar un mensaje claro en la pantalla del usuario.
