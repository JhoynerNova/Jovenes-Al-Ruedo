# RF028: Reportar artista

## Identificación
- **Módulo**: Sistema de oportunidades
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Estado**: Aprobado

## Descripción
El sistema debe permitir que una empresa reporte un perfil de artista en caso de contenido inadecuado, fraude o incumplimiento. El reporte debe llegar a los administradores junto con evidencia.

## Controles y Restricciones
- Cadía reporte debe incluir motivo obligatorio.
- Una empresa no puede reportar al mismo artista más de una vez por oferta.
- El sistema debe almacenar evidencia.

## Criterios de Aceptación
- El reporte se envía y almacena.
- El administrador recibe notificación inmediata.
- El artista reportado quedía marcado como “pendiente de revisión”.
