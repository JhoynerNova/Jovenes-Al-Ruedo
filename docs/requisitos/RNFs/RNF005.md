# RNF005: Rendimiento

## Identificación
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Categoría**: Calidad del Sistema

## Descripción
El sistema debe poder crecer en capacidad de usuarios y datos sin pérdidía de rendimiento.

## Controles y Restricciones
Arquitectura en la nube con balanceo de carga.

## Criterios de Aceptación
Añadir 1000 usuarios adicionales no debe afectar el rendimiento más de un 10%.
