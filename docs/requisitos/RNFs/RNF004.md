# RNF004: Usabilidad

## Identificación
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Categoría**: Calidad del Sistema

## Descripción
La interfaz debe ser intuitiva, clara y fácil de usar para jóvenes artistas.


## Controles y Restricciones
Debe cumplir pautasí WCAG 2.1 nivel AA.

## Criterios de Aceptación
En pruebasí con usuarios, el 80% debe completar un registro sin ayudía en menos de 2 minutos.
