# RNF001: Rendimiento

## Identificación
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Categoría**: Calidad del Sistema

## Descripción
El sistema debe cargar la página principal en menos de 3 segundos en condiciones normales y soportar hasta 1000 usuarios concurrentes.

## Controles y Restricciones
Registrar el tiempo de respuesta y generar alertasí si excede el límite.

## Criterios de Aceptación
En el 95% de lasí pruebas, el tiempo de respuesta debe ser ≤ 3s. Se cumplen lasí pruebasí de carga sin errores.
