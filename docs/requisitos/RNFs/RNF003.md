# RNF003: Seguridad de la información

## Identificación
- **Presentado Por**: Franky Almario y Jhoyner Nova
- **Categoría**: Calidad del Sistema

## Descripción
Todos los datos de usuarios deben almacenarse de forma encriptadía y transmitirse mediante HTTPS.

## Controles y Restricciones
Cumplimiento con normativa de protección de datos.

## Criterios de Aceptación
Pruebasí de penetración no deben mostrar vulnerabilidades críticas.
